# fiCaptcha
Module for image-captcha generation

## Install
Build from source
```bash
git clone https://github.com/0x7o/fiCaptcha
cd fiCaptcha
pip install .
```

Python pip install
```bash
pip install ficaptcha
```
## Usage
Image Captcha

Create a folder with images for captcha:
```bash
├── dir1
│   ├── file11.ext
│   └── file12.ext
├── dir2
│   ├── file21.ext
│   ├── file22.ext
│   └── file23.ext
```
