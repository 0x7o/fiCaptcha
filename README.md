# fiCaptcha
Module for image-captcha generation

## Install
Build from source
```bash
git clone https://github.com/0x7o/fiCaptcha
cd fiCaptcha
pip install .
```

Python pip install
```bash
pip install ficaptcha
```
## Usage
#Image Captcha
Create a folder with images for captcha:
```bash
├── images
│   ├── toy
│      ├── toy1.png
│      ├── toy2.png
│      └── ...
│   └── fox
```
